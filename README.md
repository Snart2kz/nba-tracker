# NBA Tracker – Data Science / AI Builder

## Description
Projet de suivi des performances NBA pour un joueur spécifique (exemple : LeBron James).
Analyse des points, rebonds, assists, pourcentage de tirs, minutes jouées et victoires/défaites.
Ce projet est la base pour intégrer un agent IA "coach" offline dans le futur.

## Technologies
- Python
- Pandas
- Matplotlib
- Google Colab
